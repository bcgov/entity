# vue-project-stub
## Project Setup

### Option 1: Copy this project stub
1. Install Node.js https://nodejs.org/en/
1. Copy this project folder and install from package.json.
        ```
        npm install
        ```
### Option 2. Create a new project from scratch using the Entity defaults
1. Install Node.js https://nodejs.org/en/
2. Install Vue Cli 3
        ```
        npm install -g @vue/cli
        ```
3. Move the .vuerc file to your root 
        - This file contains the defaults for a standard entities Vue project.
4. Create a new project.
        ```
        vue create my-new-project
        ```
5. Select "entities project defaults" from list of project defaults.
